Information about this project:

1. src - actual code for applciation | test - unit tests, to test robustness of application

2. Within src, the folder GradeBook holds the actual console application

3. dotnet new console - will create a new console application, the name of the C# project will take on the name of the folder its in

4. dotnet run - to run the application

5. NuGet - packages/libraries that have additional functionalities. Although .NET comes with a lot of pre-built libraries, it lacks a lot of other functionalities

6. An example of something similar to NuGet in Python is pip

7. dotnet run -- Kevin | here, the "--" is input parameters for dotnet. You need to place it to make sure that any input is for the application
